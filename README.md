# No-Phishing

A fast and secure AI powered websitewhich helps in detecting scams.

# Features
1.AI powered scam analyser that gives a risk rating.
2.Web Search verification of the output by the AI model
3.Easily understandable risk ratings
4.Blogs to spread awareness
5.Intergreated CMS with directus* along with an AI powered backup article generator
6.In-built options to contact autorities

*to be implemented as I have not yet set up the directus server
